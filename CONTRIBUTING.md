# Contributing

Co-authored by Microsoft 365 Copilot - Derek's Subscription

## Contribution Standard

Contributions must preserve:

- ITIL v5 discipline
- PMBOK v8 discipline
- no secrets
- no scope drift
- clear versioning
- updated documentation
- updated changelog

## Pull Request Requirements

- README updated if behavior changes
- CHANGELOG updated
- VERSION reviewed
- compliance workflow passes

## Completion Rule

No repo is complete without:

- README.md
- LICENSE
- LICENSE-DOCS.md
- CHANGELOG.md
- SECURITY.md
- CONTRIBUTING.md
- VERSION
- GitHub Actions compliance workflow
